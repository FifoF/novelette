Setup

Basics

Novelette requires a working TeX installation with LuaTeX and LaTeX. It does not require much else.

When you install TeXlive (or MiKTeX), you need these collections:

  • Essential programs and files
  • LaTeX fundamental packages
  • LuaTeX packages

That is a very small installation, in the TeX world. You will also need these packages, which you may already have:

xparse xpatch xifthen xstring ifmtarg fontspec
fp microtype silence changepage noindentafter

You will need hyphenation patterns for the language you will write (you do not need them for the occasional short phrase in another language). If you did not install the TeXlive collection for your language, you may install only what you need. In most cases, you need package hyphen-yourlang where yourlang is the English lowercase name of your language (french, not françcais; italian, not italiano). Disregard regional differences: canadien, suisse, français are all french.

Unless you are an advanced user who writes custom code, you do not need to read the documentation for packages other than Novelette itself. If you are an advanced user, you probably need the documentation for packages fontspec and microtype.

In some Linux operating systems, there is no lualatex binary program. Check: which lualatex. If it is not found, try this: which luahbtex. If that is not found, then your TeX installation is bad. Otherwise, navigate to the directory that has luahbtex and command: ln -s luahbtex lualatex

Required Font

Although TeX uses Latin Modern Roman as default font, it is utterly unsuitable for popular fiction, particularly when using print-on-demand. You will need to use a different font.

Novelette expects that you have Libertinus Serif installed where TeX can find it. In MiKTeX, install libertinus-fonts. In TeXlive command line:

tlmgr install libertinus-fonts

Note that other packages with libertinus in the name, are something else. You want libertinus-fonts.

If you do not use \setfont to choose something else, Novelette will use Libertinus Serif as main font. It is suited to popular fiction, although there are other fonts that look as good (or better) for this purpose.

Since Libertinus Serif has a reasonably large character set, Novelette uses it as "Plan B" when your chosen main font does not have a practicular character. For example, if your chosen main font does not have the Greek lowercase letter "pi", you can get it this way:

Pecan \altfont{\uni{03C0}} with ice cream.

Note that \uni{03C0} requests Unicode 03C0, which is pi.

If you do not choose your own main font, and do not have Libertinus Serif as default, then Novelette will use Latin Modern Roman. but it will issue a warning, and prevent using final mode.

Your Font

Novelette can use any OpenType font (extension otf) and some TrueType fonts (extension ttf). It cannot use Type 1 fonts (extension pfb, afm, pfm, vf).

The TeX Font Catalog (online web page) shows samples of all fonts that can be installed within TeX. However, you are not limited to those fonts. On most platforms, Novelette can find fonts installed in the operating system, not just the ones installed within TeX. But be sure that the fonts are licensed for your purpose.

Each font must have a license that permits embedding, for printed books, for profit. Many "free" or system fonts do not have a suitable license. In particular, DO NOT use any font that comes with the "GNU Public License" (GPL) unless it also has a "font exception" clause that allows embedding for any purpose.

Some fonts have a free version, and a paid version. You might need the paid version. Some fonts come with a program, and can only be used for files created by that program. Novelette cannot verify the license; this is your responsibilty.

Your print-on-demand book will be printed using a high-speed photocopier, not offset press (unless you strike it rich). Before committing to a font, create test files, for each Font Name. Select either letter or A4 depending on what size paper is used. Note that this is not the same as your book page size:

\documentclass{novelette}
\setfont{main}{Font Name}
\sheetfeed{letter or A4}
\begin{document}
\fillertext
\end{document}

Print the PDF on a photocopier that can print directly from digital files. Where I live, this kind of machine is often at print/ship services. Be sure to specify non-color print.

The name of the main font will be printed above the filler text. That way, you can print several test files, and know which font is which.

Pay particular attention to how dark the characters print. Fonts intended for offset presses may print too light. Pay attention to the reading experience: Does the text flow the way you expect in fiction, or does it look more like a textbook?

Only you can decide.
