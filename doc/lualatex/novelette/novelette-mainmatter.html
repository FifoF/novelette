Novelette - Design and Content of Main Matter

On Other Pages: Overview Preamble Frontmatter

What is mainmatter ?

Main Matter, or mainmatter, is your story. It includes chapters, prologue, epilogue, maps, and so forth. It may include an Introduction, if that is part of your story.

Mainmatter begins on recto Page 1. In most cases, that is where the story's written text begins, regardless of whether it is a chapter or something else. But there are exceptions:

  • If you have one or more maps on pages preceding the text, they are mainmatter. By the time the reader arrives at the opening text, the page is an odd number higher than 1.
  • If frontmatter is lengthy (don't do it), then it is customary to place a second halftitle page as the beginning of mainmatter, page 1. The opening text will have an odd page number, higher than 1.
  • If the story is divided into parts, Page 1 is the the display page announcing Part I. The story text will begin on an odd page higher than 1.

Quick Start

Here is a model for the first chapter, assumed to begin at Page 1. Each of its commands is a hyperlink within this documentation. Click to learn what the commands do, and how to choose their values.

The \null command inserts a blank line. May be repeated. Actual mainmatter opening pages will often have many \null to provide space at the top of the page, or between items on the page.

\mainmatter
\begin{opening}
\name[option]{content}
\desc[option]{content}
\init[kern]{letter}{words} and text.
\scene{text}

\mainmatter

The \mainmatter command defines the start of mainmatter. It clears to a recto page. It re-starts page numbers at arabic 1. Whether or not the page number is printed, depends on layout style.

\begin{opening} ... \end{opening}

The opening environment is used at the start of new chapters, and can also be used at the start of chapter-like portions of your book. It does not matter whether the portion is just one page.

Within an opening, you may use some commands that are not available in main text. These are \name and \desc (description). In essence, those are chapter title and subtitle (if any).

To vertically position items, use the \null command, as often as necessary, to insert blank lines.

\name[option]{Chapter Title or Number}

The \name command may only be used within the opening environment. The name is usually chapter title or number. This is not intended for more than one line of text.

The name text is pre-set at 1.8 times the size of main text.

The option is a letter: l c r o i, for alignment left, center, right, outside, or inside.

\desc[option]{Chapter Subtitle}

The \desc command may only be used in an opening environment. If used, is usually a one-line chapter subtitle. This is not intended for an extended amount of text. If necessary, you may use \desc more than once, for multiple lines. However, multiple lines would probably look better in ordinary main text.

The desc text is pre-set at 1.33 times the size of main text.

The option is a letter: l c r o i, for alignment left, center, right, outside, or inside.

\init[kern]{letter}{letters}

The \init command is optional. But if you use it for one chapter, you must uniformly use it for all of them.

It is customary to use something distinctive for the very first line of chapter text. Drop caps are out of fashion, and never were widely used for popular fiction, so Novelette does not do them.

The single letter argument is the first letter of the chapter text. If text begins with quotation marks, you may include them. This letter will be set larger than normal, so it is distinctive. It does not affect the line grid.

The plural letters are letters, perhaps a few words, that will be set with lowercase converted to small caps. Note that the letter argument already placed a letter, so do not repeat it in the letters.

After this, text is normal. Example:

\init{I}{t was dark and stormy,} but that was not unusual.

The optional kern will adjust the spacing between the enlarged letter, and the following small caps letter. It is a number, without units. Negative number moves them closer, positive farther. Rarely beyond -.2 to .2 range. Useful when the first letter is T, so you can move the following letter partly under the overhang.

If you want the first letter to be something decorative, then you will need to use fontspec syntax to define your own font command, and use it there. Novelette will not do that for you; do it yourself.

\scene{text}

The \scene command creates a scene break. If all you want is a simple blank line, use \null instead.

The text has several variations:

  • \scene{-} skips one line, and puts a long dash (about one-fourth width) in the gap
  • \scene{*} skips one line, and puts three widely-spaced asterisks in the gap.
  • \scene{text} (where text is neither - nor *) skips TWO lines, and puts the text left-aligned in the gap. The text is slightly larger than main text. It is off-grid, but this is visually best.
  • \scene{-text} (first character is hyphen) is like the above, except that the text begins with a long dash, with the first letter at indent. If your chapters are routinely divided into scenes, this method is especially attractive.

If you use a blank line instead of \scene, there may be pages where the blank line is at top or bottom. Standard practice is to use \scene{-} or \scene{*} to show that the missing line was intentional.

In all cases, it is bad layout to have just one line above or below a scene break. The only solution is to re-write.
