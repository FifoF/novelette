Novelette

This is the overview Novelette, a LuaLaTeX document class. Links to various topics are provided above.

Even if you have used TeX before, and think you know what to do, be sure to read about general setup. Novelette has many proerties that are very different from other TeX document classes. VERY different.

Properties of Novelette :

  • IS for use only with the LuaLaTeX compiler.
  • IS for popular fiction (novels and short stories).
  • IS intended for print-on-demand paper books.
  • IS able to use any modern OpenType font.
  • IS able to include black/white raster PNG images.
  • IS capable of producing PDF/X-1a:2001 files.
  • IS for users who have little prior knowledge of TeX.
  • IS simple to understand, relative to other TeX.
  • Is NOT capable of producing Ebook formats.
  • Is NOT intended for reproduction of historical works.
  • Is NOT able to include grayscale/color, text or images.
  • Is NOT able to include vector artwork (TikZ, etc.).
  • Is NOT able to use many common LaTeX commands.
  • Is NOT able to write mathematical expressions.
  • Is NOT warranted. Use at your own risk!
