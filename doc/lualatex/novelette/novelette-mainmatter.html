Novelette - Design and Content of Main Matter

On Other Pages: Overview Preamble Frontmatter

What is mainmatter ?

Main matter, or mainmatter, is your story. If something is "about the book" rather than part of the story, then it does not belong in mainmatter.

Chapters are part of the story. So is a Prologue. If you have a map of a fictional world, or a diagram of the murder scene, it is part of the story, so it belongs in mainmatter.

What about an Introduction? Novelette is intended for popular fiction, where frontmatter is short. If you have something to say (as author), put it in frontmatter, as a one or two page Author's Note.

If an Introduction is part of the story, then it belongs in mainmatter. This might be the case in the "lost manuscript" genre, in which a fictional Introduction is by a fictional archaeologist, who explains how the manuscript was discovered. The rest of the story, beginning with Chapter 1, is the fictional manuscript.

Where is Page 1 ?

Page 1, arabic numerals, is the first page of mainmatter. It is always recto. In most cases, it is the start of your story text: Chapter 1, or Prologue, or fictional Introduction.

But if you place a map prior to the start of story text, then the start of story text must be Page 3 or more (always odd). This is because the map is part of mainmatter. Page 1 may be the map, or it may be a repeat of the half-title. It may say Part I, for a multi-part book. It may be an Epigraph, but only if the Epigraph is part of the story (not from an external source such as Shakespeare).

When frontmatter is lengthy (don't do it), it is customary to repeat the half-title as Page 1, so the story begins on Page 3.

\mainmatter

The \mainmatter command defines the start of mainmatter. It clears to a recto page. It re-starts page numbers at arabic 1. Whether or not the page number is printed, depends on layout style.

\versohead{content} -and- \rectohead{content}

Commands \versohead{} and \rectohead{} apply when the layout style has a header. Otherwise, they are ignored.

These commands have no effect in frontmatter. In mainmatter, they define the content of verso and recto header text. Page numbering is automatic, so it is not affected by these commands.

You may repeat the commands throughout the book. For example, perhaps you wish to put the chapter number in the header. But be aware that most print-on-demand services require that headers contain only the book title and author name, exactly as they are registered with the distribution network. So, it is better to keep the same headers throughout the book.

The headers may be styled, using commands for small caps or italics. If you do not use these commands to define your own headers, then Novelette will automatically put the title in all-small caps above verso pages, and the author in Italics above recto pages.

Opening pages never display a header.

\begin{opening} ... \end{opening}

The opening environment is used at the start of new chapters, and can also be used at the start of chapter-like portions of your book. It does not matter whether the portion is just one page.

Within an opening, you may use some commands that are not available in main text. These are \name and \desc (description). In essence, those are chapter title and subtitle (if any).

To vertically position items, use the \null command, as often as necessary, to insert blank lines.

\name[option]{Chapter Title or Number}

The \name command may only be used within the opening environment. The name is usually chapter title or number. This is not intended for more than one line of text.

The name text is pre-set at 1.8 times the size of main text.

The option is a letter: l c r o i, for alignment left, center, right, outside, or inside.

\desc[option]{Chapter Subtitle}

The \desc command may only be used in an opening environment. If used, is usually a one-line chapter subtitle. This is not intended for an extended amount of text. If necessary, you may use \desc more than once, for multiple lines. However, multiple lines would probably look better in ordinary main text.

The desc text is pre-set at 1.33 times the size of main text.

The option is a letter: l c r o i, for alignment left, center, right, outside, or inside.
